## 使用方式

1. 安装 [Kristal](https://github.com/KristalTeam/Kristal) 引擎。
2. 将本仓库克隆到 Kristal 的 `mods/` 目录下：

   ```bash
   cd Kristal/mods
   git clone https://github.com/Bli-AIk/krisis_knightmare.git
   ```

3. 启动 Kristal，在模组选择中选择 **krisis_knightmare**。
